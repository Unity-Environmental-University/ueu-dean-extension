Installation

Get it running
in five minutes.

Chrome doesn't install this from the store. You load it directly from a downloaded folder. Here's how.

01

Download the latest release

Go to the releases page and download the .zip file from the most recent release.

The file will be named something like ueu-dean-extension-v0.2.0.zip.

02

Unzip it somewhere stable

Double-click the zip to unzip it. Move the resulting folder somewhere you won't accidentally delete it — your Documents folder works well.

Chrome loads the extension from this folder every time it starts. If you move or delete the folder later, the extension will stop working.

03

Open Chrome's extension manager

In Chrome, go to chrome://extensions — type that directly into the address bar and press Enter.

04

Turn on Developer mode

In the top-right corner of the extensions page, flip the Developer mode toggle on. A row of new buttons will appear.

05

Click "Load unpacked"

Click the Load unpacked button, then select the folder you unzipped in step 2.

The extension will appear in your list.

06

Pin it to your toolbar

Click the puzzle-piece icon in Chrome's toolbar, find Dean Tools, and click the pin icon so it stays visible.

That's it. Open any Stratus case and the U button will appear in the bottom-right corner of the page.

Updating

Two minutes
to update.

When a new version is available, you'll get an email. Here's how to install it.

01

Download the new release

Go to the releases page and download the .zip file from the newest release.

02

Replace the old folder

Unzip the download and replace the old extension folder with the new one. Use the same location — if you put the original in Documents, put the new one there too.

You can delete the old folder first, or just overwrite it. Either way works.

03

Reload in Chrome

Go to chrome://extensions, find UEU Dean Tools, and click the reload icon (the circular arrow). That's it.

You can also press Alt+Shift+R on any Salesforce page to reload the extension without opening the extensions page.
